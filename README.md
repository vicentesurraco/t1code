## What it is

`t1code` is a terminal-first fork of T3 Code.

## Install

```bash
npx @maria_rcks/t1code
```

## What you need

- the required CLI tooling installed
- provider auth already set up
- macOS-first for now

## Development

```bash
bun install
bun run dev:tui
```

For the required checks:

```bash
bun run fmt
bun run lint
bun run typecheck
```
